# Alpha_server
Create a local PHP server

# Alpha Server Setup Guide

Follow these steps to get your local server running:

1. **Deploy Files**: Place your PHP files inside the `www` folder.
2. **Start the Engine**: Open the **Alpha Server** application and click **"Start Server"**.
3. **View Website**: Click **"Open Site"** to automatically launch your site in the browser.

 **Important**: Do **not** close the PHP pop-up window, as it is required to keep the server active.
